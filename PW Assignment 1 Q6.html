List of HTML Tags

  1. <html>: The root element of an HTML document that defines the entire web page.
  2. <head>: Contains metadata about the HTML document, such as the title and linked stylesheets or scripts.
  3. <body>: Represents the content of an HTML document, including text, images, links, and other elements.
  4. <h1> to <h6>: Defines headings of different levels, with <h1> being the highest and <h6> being the lowest.
  5. <p>: Represents a paragraph of text.
  6. <a>: Creates a hyperlink that allows users to navigate to another web page or a specific location within the current page.
  7. <img>: Embeds an image into the HTML document.
  8. <ul>: Defines an unordered (bulleted) list of items.
  9. <ol>: Defines an ordered (numbered) list of items.
  10. <li>: Represents an item in a list (<ul> or <ol>).
  11. <table>: Represents tabular data organized in rows and columns.
  12. <tr>: Defines a row in a table.
  13. <td>: Represents a data cell in a table.
  14. <div>: Defines a division or a container that helps structure the content and apply styles.
  15. <span>: Represents an inline element used for grouping and applying styles to a specific portion of text or elements.
