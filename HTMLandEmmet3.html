Q3:- Write a short note on tags, elements, and attributes along with relevant examples.

1. Tags:

In HTML tags are used to define the structure and content of a web page. Tags are enclosed within angle brackets ("<>" and "") and indicate the beginning and end of an element . Elements can be thought of as building blocks of a web page, and they are defined by the tags used.

Example:-

<P>Lorem ipsum dolor sit amet. </ tag >

<p>In the example above, the <p> tags define a paragraph element. The text "Lorem ipsum dolor sit amet.." is the content of the paragraph, and it is enclosed within the opening and closing tags.




2. Emmet:

Elements are the basic units of HTML and represent different types of content or components on a web page. They are composed of opening tags, closing tags, and the content in between. Elements can be nested within each other to create a hierarchical structure.

Example:-

<div>
<h1>Welcom to my Website </h2>
<p>This is my paragraph </p>
</div>

In the example above, the < div > element is a container that holds the heading < h1 > and paragraph < p > elements. The content inside the < div > forms a section of the web page.




3. Attributes

Attributes provide additional information about HTML elements. They are added within the opening tag of an element and consist of a name and a value, separated by an equals sign. Attributes can be used to modify the behavior or appearance of an element.

Example

<a href="https://www.example.com">Visit Example.com </a>

In the example above, the < a > element is an anchor element used for creating links. The href attribute specifies the URL that the link should navigate to. In this case, it points to "https://www.example.com". The text "Visit Example.com" is the content of the anchor element and serves as the clickable link.
