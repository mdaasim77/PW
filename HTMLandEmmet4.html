Q.4:- List out any 3 tags we learned in this modude and give a brief explanation about the tags.



1. <h1 > to <h6> (Headings):

Headings are used to define the titles or headings of sections on a web page. There are six levels of headings, ranging from <h1> (the highest level) to <h6> (the lowest level). The importance or hierarchy of the heading decreases as the level number increases.

Example

<h1> This is heading 1 <h1>

<h2> This is heading 2 <h2>

<h3> This is heading 3 <h3>




2. <img> (Image):

The <img> tag is used to insert images into a web page. It does not have a closing tag and requires the src attribute to specify the image source ( URL or file path) and the alt attribute to provide alternative text for accessibility purposes. The alt attribute describes the image in case it cannot be displayed or accessed.

Example

<img src="path/to/image.jpg" alt="A beautiful landscape">




3. <a> (Anchor):

The <a> tag creates hyperlinks or anchor links in HTML. It is used to link one web page to another or to a specific secti on within the same page. The hr ef attribute is used to specify the destination URL, and the con tent within the opening and closin g tags serves as the clickable text.

Example

<a href="https://www.example.com">V isit Example.com</a>
