Full Stack Web Development Tech Stack

HTML
HTML (Hypertext Markup Language) is used for creating the structure and content of web pages.
JavaScript
JavaScript is a programming language that adds interactivity and dynamic behavior to web pages.
Node.js
Node.js is a JavaScript runtime environment that allows server-side execution of JavaScript.
Express.js
Express.js is a minimalist web application framework for building server-side applications with Node.js.
MongoDB
MongoDB is a NoSQL database system used for storing and retrieving data in a structured format.
