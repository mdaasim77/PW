Most Common Text Formatting Tags in HTML

  1. <b> or <strong>
    • Use: Specifies bold text.
    • Best Practices: Use <strong> for emphasizing important content with semantic meaning, and use CSS for styling purposes.
  2. <i> or <em>
    • Use: Specifies italicized text.
    • Best Practices: Use <em> for emphasizing content with semantic meaning, such as emphasizing a word or phrase within a sentence, and use CSS for styling purposes.
  3. <u>
    • Use: Specifies underlined text.
    • Best Practices: Avoid using underlined text for regular content as it may be confused with hyperlinks. Use CSS for styling purposes.
  4. <sup>
    • Use: Specifies superscript text.
    • Best Practices: Use for displaying mathematical notations, footnotes, or references.
  5. <sub>
    • Use: Specifies subscript text.
    • Best Practices: Use for displaying chemical formulas, mathematical notations, or footnotes.
